## 💡 Today I Learned

### **1. PriorityQueue (우선순위 큐) 자료구조**
- **개념**: 일반적인 FIFO 구조가 아닌, 우선순위가 높은 데이터가 먼저 나가는 자료구조.
- **내부 원리**: **Heap(힙)** 알고리즘을 사용하여 삽입/삭제 시 $O(\log N)$의 시간 복잡도를 가짐.
- **주요 메서드**:
    - `peek()`: 최우선 요소 확인 (삭제 X)
    - `poll()`: 최우선 요소 추출 및 삭제
    - `offer()`: 데이터 삽입 및 자동 정렬

### **2. Java 정렬 인터페이스 활용**
- **Comparable**: 객체의 '기본 정렬 기준'을 세울 때 사용. (`this`와 `other` 비교)
- **정렬 로직**: `this - other` 결과가 음수이면 내가 앞순서. 실무적 안정성을 위해 `Integer.compare()` 사용 권장.

## 📝 문제 풀이

### **[백준 11000] 강의실 배정**
- **난이도**: Gold 5
- **유형**: Greedy, Sorting, PriorityQueue
- **문제 링크**: [백준 11000번](https://www.acmicpc.net/problem/11000)

#### **풀이 전략**
1. **정렬**: 수업을 시작 시간 기준으로 오름차순 정렬하여 시간 흐름대로 배치함.
2. **최소 힙 관리**: 현재 사용 중인 강의실들의 '종료 시간'만 PQ에 담아 관리.
3. **그리디 선택**: 
   - 가장 빨리 비는 시간(`pq.peek()`) $\le$ 다음 수업 시작 시간 이면? -> 강의실 재사용 (`poll` 후 `offer`)
   - 아니면? -> 새 강의실 추가 (`offer`만 수행)

#### **작성 코드 (Java)**
```java
import java.io.*;
import java.util.*;

class Main {
    static class Lecture implements Comparable<Lecture> {
        int start, end;
        Lecture(int start, int end) { this.start = start; this.end = end; }

        @Override
        public int compareTo(Lecture other) {
            if (this.start == other.start) return Integer.compare(this.end, other.end);
            return Integer.compare(this.start, other.start);
        }
    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int N = Integer.parseInt(br.readLine());
        List<Lecture> lectures = new ArrayList<>();

        for (int i = 0; i < N; i++) {
            StringTokenizer st = new StringTokenizer(br.readLine());
            lectures.add(new Lecture(Integer.parseInt(st.nextToken()), Integer.parseInt(st.nextToken())));
        }

        Collections.sort(lectures);
        PriorityQueue<Integer> pq = new PriorityQueue<>();
        pq.offer(lectures.get(0).end);

        for (int i = 1; i < N; i++) {
            if (pq.peek() <= lectures.get(i).start) pq.poll();
            pq.offer(lectures.get(i).end);
        }
        System.out.println(pq.size());
    }
}
```
